# NotesApp

A full-stack Notes application built as a technical assessment.

## Tech Stack

### Backend
- ASP.NET Core 8 Web API
- Dapper
- SQL Server

### Frontend
- Vue 3
- TypeScript
- Tailwind CSS
- Pinia
- Axios

## Features

### Authentication

- User Registration
- User Login
- Refresh Token
- Logout
- Swagger JWT Authorization
